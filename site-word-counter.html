Site Word Counter

October 31, 2019 TAGS projects / flask / nltk

I followed this tutorial (part 1 and part 2) and deployed the code to heroku (Application is live here).

This was a fun project because it uses Flask, Beautiful Soup and NLTK. The user enters a url and the code fetches the content from that url and returns an alphabetical list of the words used on that page, as well as a count for each word. The code uses requests to fetch the content from the url, BeautifulSoup to parse the content and remove the html tags, and NLTK to count the words.

Unfortunately, I did not upload the code to GitHub or BitBucket before my main computer crashed, but I did manage to deploy it to Heroku.
